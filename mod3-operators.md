# Operators

Omm, like most programming languages, uses operators to perform tasks. The list of Omm's operators are shown here:

| Name | Function |
|:----:|:---------:|
|+|Addition and string concatenation|
|-|Subtraction|
|*|Multiplication|
|/|Division|
|%|Modulus|
|^|Exponentiation|
|::|Indexing|
|=|Assignment|
|:=|Declaration|
|->|Type casts|
|==|Equality|
|!=|Inequality|
|<|Less than|
|>|Greater than|
|<=|Less than or equal|
|>=|Greater than or equal|
|:|Synchronous Call|
|?|Asynchronous Call|
|!|Not|
|&|And|
|\||Or
